Collection Integration Process

Phase 1: Collection Inventory

Complete detailed inventory of physical and digital collections from both libraries

Phase 2: Overlap Analysis

Identify duplicates and evaluate using retention criteria matrix

Phase 3: Retention Decisions

Make keep/discard decisions for physical materials and database subscriptions

Phase 4: Catalog Migration

Consolidate catalog records and implement unified catalog system

Phase 5: Physical Move Planning

Develop space requirements and moving timeline for physical collections
